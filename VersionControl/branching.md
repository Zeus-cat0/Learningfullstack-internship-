# Git Branching

## What is a Branch?

A branch is a separate line of development that allows you to work on new features without affecting the main project.

## Why use Branches?

- Work on new features safely
- Fix bugs independently
- Experiment without affecting main code
- Collaborate with team members

## Basic Commands

git branch
git branch feature-login
git switch feature-login
git switch main
git merge feature-login
git branch -d feature-login
