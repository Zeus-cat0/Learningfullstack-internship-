# Version Control

## What is Version Control?
Version control is a system that tracks changes made to files over time. It allows developers to go back to previous versions, collaborate with others, and manage projects safely.

## Why use Git?
- Tracks file changes
- Keeps project history
- Allows collaboration
- Supports branching and merging
- Makes backups easy
